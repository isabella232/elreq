What is Zaima?

Intended audience: anyone who has heard of Zaima annotations and would like to know a little more about it.

Question

What are 'Zaima' annotations?

Quick answer

Zaima is a name for small annotations that are rendered alongside base text. This is especially useful for Japanese and other East Asian content (ruby may be called furigana in Japanese).

The HTML5 specification provides a way to mark up ruby text, and CSS can be used to fine tune of the positioning of ruby text.
